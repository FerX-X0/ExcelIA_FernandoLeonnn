# Dashboard Inteligente con Excel + IA

Autor: Fernando Leon

## Problema

Los estudiantes frecuentemente no saben en qué categorías gastan más
dinero cada mes.

## Objetivo

Analizar gastos personales utilizando Excel y herramientas de
inteligencia artificial.

## Archivos

-   dashboard_excel_ia_gastos.xlsx
-   informe_proyecto_excel_ia.docx

## Resultados

Se creó un dashboard que muestra los gastos por categoría y por mes,
permitiendo visualizar tendencias y mejorar la toma de decisiones
financieras.
